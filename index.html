Возможный список запросов к базе данных:

1. Выбирает из таблицы ИЗДАНИЯ информацию о доступных для подписки газетах, название которых начинается с буквы «П».

2. Выбирает из таблиц информацию о подписчиках, проживающих на улице «Садовая», которые оформили подписку на издание с индексом «12123».

3. Выбирает из таблицы ПОЛУЧАТЕЛИ информацию о подписчиках, проживающих на улице «Садовая» в домах с номерами 2,7,8.

4. Выбирает из таблицы ИЗДАНИЯ информацию об издании с некоторым индексом. Значение индекса вводится при выполнении запроса.

5. Выбирает из таблицы ИЗДАНИЯ информацию обо всех изданиях, для которых цена 1 экземпляра есть значение из некоторого диапазона. Нижняя и верхняя границы диапазона задаются при выполнении запроса.

6. Вычисляет для каждой оформленной подписки ее стоимость без доставки и без НДС. Включает поля Индекс издания, Наименование издания, Цена 1 экземпляра, Дата начала подписки, Срок подписки, Стоимость подписки.

7. Выполняет группировку по полю Вид издания. Для каждого вида вычисляет среднюю цену 1 экземпляра.

8. Выполняет группировку по полю Улица. Для каждой улицы вычисляет количество подписчиков, проживающих на данной улице.
